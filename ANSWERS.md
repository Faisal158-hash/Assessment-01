## Q1: How to run
Open the project folder and run index.html in any browser.
No installation is required.


## Q2: Stack & Design
I used HTML, CSS, and JavaScript because it is simple and fast for this task.
Design Decision 1:
I made the timer large so users can easily see time from a distance.
Design Decision 2:
I used different colors for focus (red) and break (green) so users quickly understand the current mode.


## Q3: Responsive & Accessibilit
On mobile (360px), the layout becomes vertical and buttons stack.
On laptop (1440px), everything is centered with more spacing.
Accessibility:
I used clear contrast colors for readability.
Skipped:
Keyboard navigation was not fully implemented due to time.


## Q4: AI Usage
I used ChatGPT to help generate timer logic.
The AI gave a basic countdown, but I modified it to support switching between focus and break modes automatically.



## Q5: Honest Gap
The UI design is basic.
With more time, I would improve animations and add better visual feedback for transitions.
